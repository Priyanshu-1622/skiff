# Contributing

Thanks for taking a look. This is a small project I work on solo, so things are pretty informal.

If you find a bug or want a feature, open an issue first so we can talk it through before you write code. PRs are welcome for anything we've discussed.

A couple of things:
- TypeScript strict mode is on, please keep it that way.
- Run `pnpm dev` to get both the API and web app running locally.

That's it. Don't overthink it.
